# ✅ Task List Manager

A simple and efficient command-line Python application that helps you manage your daily tasks. Easily add, view, complete, and delete tasks using an interactive terminal interface.

---

## 🚀 Features

- 📋 Add new tasks
- ✅ Mark tasks as completed
- ❌ Delete tasks
- 📄 View all tasks 

---

## 🛠️ Tech Stack

- **Language:** Python 3.x
